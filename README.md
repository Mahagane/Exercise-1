# Exercise-1
Variables, Collections &amp; Conditionals
Introduction to the Assignment

This assignment teaches the basic foundations of Python programming. It focuses on:

Variables
Data types
Collections (Lists and Dictionaries)
Conditional statements
Logical operators
Working with student data

The assignment uses examples of students, their scores, ages, subjects, and pass/fail status.

2. Main Objectives of the Assignment

The objectives are:

Understand how to store data using variables
Learn different Python data types
Learn how collections store multiple values
Understand how lists and dictionaries work
Learn how to use conditional statements (if, elif, else)
Learn how logical operators work (and, or, not)
Practice solving simple real-world problems using Python
